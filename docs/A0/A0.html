Phase 6: The Demo

The astute observer likely would have noticed that there is no potentiometer in my circuit schematic, alas I lost all of my potential in exchange for something even more tanglible.

Remember how I said I got the "Nano Sense 33 BLE Rev 2" (what is with the Arduino names) because it has all the sensors embedded on the board? Well I was playing around with those at some point (probably while helping an elderly woman cross the street) and I had the idea to control the voltage signal with a sensors input. After experimenting a bit with a few of them, I decided the best one to use would be the APDS9960 proximity sensor since the gesture of moving something closer to a source felt like a natural metaphor for increasing power.

A handy thing about the Arduino library for this sensor is that it returns a value between 0 and 255, which as I recently learned is the scale used by the digital pins for pulse width modulation (PWM). PWM is how you digitally control voltage, since a digital signal is either max output or zero output, you just modulate that signal at a certain frequency to get an analog-like signal.

The result? Something I like to call the Vibro-ProxiLite-9000

Phase 7: We're Rich.

I forgot to mention one thing, the code for this project is incredibly simple, like I really couldn't believe how chill it was to set this stuff up in the Arduino IDE. Of course, it's only chill because I'm using the API for the APDS9960 sensor, I've had to reconfigure similar libraries for IMU's in the past and that can be a massive pain...

Anyway, here's the code for the project:

                        
#include <Arduino_APDS9960.h>

const int motorpin = 2;

void setup() {

  Serial.begin(9600);
  pinMode(motorpin, OUTPUT);
  
  while (!Serial);
  if (!APDS.begin()) {
            Serial.println("Error initializing APDS9960 sensor!");
      }
}

void loop() {

     if (APDS.proximityAvailable()) {
          int proximity = APDS.readProximity();
          Serial.print("Proximity: ");
          Serial.println(proximity);
          
          analogWrite(motorpin, proximity);
     }
}
